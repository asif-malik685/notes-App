# 📝 Notes App

A simple and modern Notes App built using **HTML, CSS, and JavaScript** with **Local Storage** support.  
It allows users to add, delete, and store notes directly in the browser.

---

## 🚀 Features
- Add new notes easily
- Delete notes with one click
- Data saved in browser using Local Storage
- Notes remain after page refresh
- Clean and responsive UI
